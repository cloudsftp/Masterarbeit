\subsection{Archetypal Model}
\label{app:impl.model.arch}

\Cref{lst:app.impl.arch.cpp} shows the code used for the simulations of the archetypal model.
As one can see, this code is a lot shorter than the implementation of the original model in \Cref{lst:app.impl.orig.cpp}.
To simulate the halved archetypal model, one simply changes the lines 47 and 48 to use \texttt{n} instead of \texttt{2 * n}.
